
# Elemes.id

## Description
This is my frontend test project for Elemes.id.




## Run Locally

Clone the project

```bash
  git clone https://github.com/PramudyaReynaldi/Elemes.id.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
